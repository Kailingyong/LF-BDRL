# LF-BDRL

This repository is the official implementation of the paper "Light Field Image Blind Super-Resolution via Degradation Representation Learning" published in Pattern Recognition (PR) 2026.

## Pretrained Models

The pretrained checkpoints of our model are available for download via Baidu Netdisk. 

You can download the weights file from [this download link](https://pan.baidu.com/s/1qob6mj0Kke0ZicalytHS2g?pwd=31t3) (Extraction Code: `31t3`).
